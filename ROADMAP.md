# Roadmap

Post-v1.0.0 direction. Timelines are indicative, not commitments.

## Near term (v1.1)

- [ ] Hosted docs site (replace GitHub redirect on marketing website)
- [ ] Weblate project live with automated locale PR sync
- [ ] Expanded plugin marketplace curation
- [ ] Performance profiling for large monorepo tasks

## Medium term (v1.x)

- [ ] Additional provider adapters and routing policies
- [ ] Improved VM image reproducible build pipeline
- [ ] Mobile companion read-only task status (optional)
- [ ] Community theme variants in design system

## Long term (v2+)

- [ ] Pluggable execution backends beyond bundled Linux VM
- [ ] Federated Aura Cloud instances
- [ ] Enterprise SSO and policy packs (optional modules)

## How to influence the roadmap

- Vote and discuss in [GitHub Discussions → Ideas](https://github.com/hbx12/aura-work/discussions/categories/ideas)
- Submit focused PRs for items marked as help-wanted in Issues

See [CHANGELOG.md](./CHANGELOG.md) for shipped history.
