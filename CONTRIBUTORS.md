## How to get listed

1. Make a contribution (code, docs, translations, design, bug reports, etc.).
2. A maintainer will add you, or you can open a PR adding yourself using the [All Contributors bot](https://allcontributors.org/docs/en/bot/usage):

```
@all-contributors please add @<your-username> for code, doc
```

## Contribution types

| Emoji | Type         | Emoji | Type        |
|-------|--------------|-------|-------------|
| 💻    | Code         | 📖    | Docs        |
| 🌍    | Translation  | 🎨    | Design      |
| 🐛    | Bug report   | 🔌    | Plugin/MCP  |
| ✅    | Tutorial     | 🤔    | Ideas       |
