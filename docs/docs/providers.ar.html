المزوّدون المحليون

يدعم Aura Work 10 مزوّدي ذكاء اصطناعي تشمل النماذج السحابية والمحلية. كل مزوّد له محول خاص يدير المصادقة واكتشاف النماذج. المزوّدون المحليون لا يرسلون بيانات خارج جهازك أبداً.

ما هي المزوّدات؟

المزوّدات هي الجسر بين Aura Work ونماذج الذكاء الاصطناعي. كل مزوّد ينفّذ واجهة موحدة تتعامل مع:

  • المصادقة — مفاتيح API، رموز OAuth، أو اتصالات محلية
  • اكتشاف النماذج — اكتشاف تلقائي للنماذج المتوفرة
  • إكمالات المحادثة — إرسال الطلبات واستلام الردود
  • تتبع الاستخدام — عدّ الرموز (tokens) وتقدير التكلفة

Aura Cloud Models

3 نماذج سحابي

نماذج مستضافة عبر API Aura. تشمل Aura Fast و Aura Coder و Aura Premium.

🔑 سجّل الدخول بحساب Aura Cloud من الإعدادات ← المزوّدون ← Aura Cloud. لا حاجة لمفتاح API منفصل — يتم استخدام رمز الجلسة المرتبط بحسابك.

Anthropic

2 نماذج سحابي

Claude Sonnet 4 و Claude 3.5 Haiku. قدرات استدلال وتوليد كود ورؤية من الطراز الأول.

🔑 أنشئ مفتاح API من console.anthropic.com، ثم الصقه في الإعدادات ← المزوّدون ← Anthropic واضغط 'تحقق'.

OpenAI

2 نماذج سحابي

GPT-4o و GPT-4o mini. نماذج لغوية معيارية مع دعم واسع لاستدعاء الأدوات.

🔑 أنشئ مفتاح API من platform.openai.com/api-keys. يدعم أيضاً حسابات GitHub Copilot عبر تسجيل الدخول بـ OAuth.

Google Gemini

2 نماذج سحابي

Gemini 2.0 Flash و Gemini 2.5 Pro. استدلال قوي مع نوافذ سياق ضخمة.

🔑 أنشئ مفتاح API من Google AI Studio (aistudio.google.com)، ثم أضفه في إعدادات المزوّد.

DeepSeek

2 نماذج سحابي

DeepSeek V3 و R1. نماذج فعالة من حيث التكلفة مع استدلال تنافسي.

🔑 أنشئ مفتاح API من platform.deepseek.com. الأسعار منخفضة جداً مقارنة بالمزودين الآخرين.

Ollama

نماذج متعددة محلي

محلي بالكامل. شغّل أي نموذج من مكتبة Ollama. لا حاجة لمفتاح API ولا تغادر بياناتك جهازك.

🔑 ثبّت Ollama من ollama.ai، شغّل ollama serve، ثم اسحب نموذجاً بـ ollama pull llama3. لا حاجة لمفتاح API.

OpenAI Compatible

نماذج متعددة محلي

نقاط نهاية مخصصة تحاكي واجهة OpenAI. استخدم مع أي خدمة متوافقة.

🔑 أدخل عنوان URL الأساسي (base URL) للخادم المتوافق مع OpenAI، ومفتاح API إن وجد.

Minimax

1 نماذج سحابي

نماذج Minimax. مفتاح API مطلوب.

🔑 أنشئ مفتاح API من منصة Minimax الرسمية وأضفه في إعدادات المزوّد.

Qwen

2 نماذج سحابي

نماذج Qwen (Plus و Turbo). مفتاح API مطلوب.

🔑 أنشئ مفتاح API من منصة Alibaba Cloud DashScope وأضفه في إعدادات المزوّد.

LM Studio

نماذج متعددة محلي

محلي بالكامل. اتصل بخادم LM Studio المحلي. لا حاجة لمفتاح API.

🔑 ثبّت LM Studio من lmstudio.ai، حمّل نموذجاً، وشغّل الخادم المحلي من تبويب 'Local Server'. لا حاجة لمفتاح API.

المزودون المدعومون

يدعم Aura Work 10 مزودين لضمان المرونة القصوى. لكل مزود نقاط قوته:

السحابيون (Cloud)

  • OpenAI - GPT-4o, GPT-4.5, o1, o3. أداء متميز في البرمجة والتحليل
  • Anthropic - Claude Opus, Sonnet, Haiku. سياق طويل ورؤية حاسوبية
  • Google Gemini - Gemini 2.5 Pro, Flash. متعدد الوسائط ونماذج خفيفة
  • DeepSeek - DeepSeek-V3, R1. أداء قوي بتكلفة منخفضة
  • xAI Grok - Grok-3. نماذج متطورة من xAI

المحليون (Local)

  • Ollama - LLama, Mistral, Gemma, Qwen. خصوصية كاملة
  • LM Studio - واجهة رسومية لتشغيل النماذج محليا
  • LocalAI - OpenAI-compatible API محلية
الأمان والموثوقية

حماية بيانات الاعتماد

تُشفَّر مفاتيح API باستخدام الخزنة المرتبطة بالجهاز قبل التخزين. تستخدم الخزنة:

  • Windows — DPAPI (واجهة حماية البيانات)
  • macOS — Keychain
  • Linux — Secret Service (GNOME Keyring / KWallet)

بيانات الاعتماد لا تُسجَّل أبداً، ولا تُعرَض للوكيل، ولا تُدرَج في سجلات التدقيق.

🔄 التراجع وإعادة المحاولة

إذا فشل مزوّد، يمكن لـ Aura Work إعادة المحاولة تلقائياً مع بديل:

  1. 1. يفشل المزوّد الأساسي (حد المعدل، مهلة، خطأ)
  2. 2. يفحص النظام عن مزوّدين احتياطيين في إعداداتك
  3. 3. إذا وُجد احتياطي، يعيد المحاولة مع المزوّد البديل
  4. 4. إذا لم يوجد احتياطي، يُعلمك ويطلب تدخلاً يدوياً

💡 نصائح لتحسين التكلفة

  • استخدم سياسة التكلفة أولاً للمهام الروتينية
  • أعدّ Ollama للتطوير والاختبار (مجاني)
  • استخدم DeepSeek لمهام الكود (أرخص من OpenAI/Anthropic)
  • راقب الاستخدام في لوحة التحكم لتحديد الأنماط المكلفة
  • عيّن حدوداً للرموز لكل مهمة لمنع التكاليف المتزايدة
التكوين

كيفية تكوين المزودين

استخدم الامر التالي لتكوين مزود:

aura config set providers.openai.apiKey sk-...
aura config set providers.anthropic.apiKey sk-ant-...

للمزودين المحليين:

aura config set providers.ollama.baseUrl http://localhost:11434
aura config set providers.ollama.model llama3.2

كل المفاتيح تخزن مشفرة باستخدام تشفير على مستوى النظام (Keytar على macOS، DPAPI على Windows، libsecret على Linux).
