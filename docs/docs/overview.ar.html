ما هي Aura Work ولماذا وُجدت

Aura Work هي منصة وكلاء ذكاء اصطناعي مكتبية متعددة المزوّدين مبنية للمطورين الذين يريدون مساعدة ذكاء اصطناعي محلية أولاً ومقيدة بالأذونات وقابلة للاستضافة الذاتية. تدعم 10 مزوّدي ذكاء اصطناعي فورياً، مع محرك توجيه ذكي يختار النموذج المناسب لكل مهمة.

على عكس الأدوات السحابية فقط، يحتفظ Aura Work ببياناتك على جهازك. يستخدم هندسة إضافات مع تنفيذ معزول وتخزين مشفر للبيانات الاعتمادية وتحكم دقيق في الأذونات.

تطبيق سطح المكتب

غلاف Tauri 2 يستضيف واجهة React 19. يدير النوافذ وعلبة النظام والقوائم والتواصل مع الخدمات المساعدة.

الخدمات المساعدة

8 خدمات خلفية لتنفيذ الوكلاء وأتمتة المتصفح والعزل الافتراضي والمزامنة السحابية والمزيد.

محرك الوكيل

تنسيق متعدد الوكلاء مع أدوار المخطط والمنفذ والمراجع والأمان. يدعم أدوات مخصصة وخوادم MCP.

محرك التوجيه

موجه نماذج ذكي عبر 10 مزوّدين مع 5 سياسات توجيه: الجودة أولاً، التكلفة أولاً، الخصوصية أولاً، محلي فقط، أو يدوي.

لماذا Aura Work

لماذا تختار Aura Work

معظم أدوات الذكاء الاصطناعي إما سحابية فقط أو محدودة بمزود واحد. Aura Work يحل كلا المشكلتين:

  • متعدد المزودين - استخدم OpenAI و Anthropic و Gemini و DeepSeek و Ollama من واجهة واحدة
  • محلية أولا - بياناتك تبقى على جهازك، مشفرة بمفاتيح مرتبطة بالجهاز
  • مقيدة بالأذونات - كل إجراء يتطلب موافقة صريحة
  • قابلة للاستضافة الذاتية - شغل خادم المزامنة السحابية الخاص بك
  • قابلة للتوسيع - أضف قدرات عبر المهارات وخوادم MCP والإضافات

القدرات الأساسية

Aura Work أكثر من مجرد واجهة محادثة. إنه بيئة تطوير كاملة تشمل:

  • محرك المهام - تنفيذ مهام متعددة الخطوات مع التخطيط والتنفيذ والمراجعة
  • عمليات الملفات - قراءة وكتابة وبحث وإدارة الملفات
  • تكامل Git - الحالة والفرق والالتزام وإدارة الفروع
  • أتمتة المتصفح - تصفح المواقع واستخراج البيانات
  • استخدام الحاسوب - التحكم بتطبيقات سطح المكتب
  • صندوق الرمل - تنفيذ أوامر Shell في بيئات WSL2 أو حاويات
  • المزامنة السحابية - مزامنة E2EE عبر الأجهزة
