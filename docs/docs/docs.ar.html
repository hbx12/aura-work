روابط سريعة

ابدأ من هنا

سواء كنت مستخدماً جديداً أو مطوراً يريد المساهمة في الكود المصدري، إليك أسرع الطرق للوصول إلى ما تحتاجه بالضبط:

👋 مستخدم جديد؟

ابدأ بصفحة نظرة عامة لفهم فلسفة Aura Work، ثم انتقل إلى البداية السريعة لتثبيت التطبيق وتشغيل أول مهمة خلال دقائق.

🧑‍💻 مطور أو مساهم؟

راجع صفحة الهندسة لفهم بنية النظام متعددة العمليات، ثم المهارات و بروتوكول السياق (MCP) لإضافة قدرات جديدة للوكيل.

🔐 مهتم بالخصوصية والأمان؟

اطلع على الأذونات و التوجيه لمعرفة كيف يبقى Aura Work محلي البيانات وآمناً بشكل افتراضي دون أي تنازلات.

⌘ تفضل الطرفية (Terminal)؟

راجع صفحة واجهة الأوامر للتحكم الكامل في Aura Work من سطر الأوامر — إنشاء المهام ومتابعتها عن بُعد.
