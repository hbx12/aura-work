التثبيت

المتطلبات الأساسية

  • نظام التشغيل: Windows 10+ أو macOS 12+ أو أي توزيعة Linux حديثة
  • مساحة تخزين: ~200 MB للتطبيق + نموذج Ollama للاستخدام المحلي
  • اتصال بالإنترنت: مطلوب للمزوّدين السحابيين فقط (غير مطلوب لـ Ollama/LM Studio)

الخطوات

1. تنزيل من إصدارات GitHub

تصفح صفحة الإصدارات وحمّل أحدث إصدار لنظام تشغيلك.

2. فك الضغط وشغّل Aura Work

فك ضغط الملف وشغّل التطبيق. سيظهر في شريط المهام (أو علبة النظام) عند بدء التشغيل.

3. إعداد مزوّد

اذهب إلى الإعدادات → المزوّدون. أضف مفتاح API لأي مزوّد أو ابدأ Ollama محلياً مجاناً. يتم التحقق من كل مزوّد تلقائياً.

4. تشغيل أول مهمة

اكتب أمراً مثل: "أنشئ مكون React جديد يجلب البيانات من API". يخطط الوكيل وينفذ ويعرض النتائج.

5. استكشف

جرّب المهارات من السجل، ثبّت خوادم MCP، اضبط سياسات التوجيه، أو استخدم CLI للتحكم من الطرفية.

التثبيت المتقدم

خيارات التثبيت الاخرى

باستخدام Homebrew (macOS و Linux)

brew install aura-work

باستخدام Winget (Windows)

winget install aura-work

البناء من المصدر

git clone https://github.com/hbx12/aura-work.git
cd aura-work
npm install
npm run build
npm link

يتطلب Node.js 18+. يدعم Windows و macOS و Linux مع حزم منفصلة لكل منصة.

الاستخدام الاول

تجربتك الاولى

بعد التثبيت، جرب هذه الاوامر للتعرف على Aura Work:

aura --help
aura --version
aura config list
aura --dry-run "انشئ مشروع Node.js جديد"
