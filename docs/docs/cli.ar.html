واجهة الأوامر

واجهة الأوامر للتحكم عن بعد في الوكيل وسير العمل القابلة للبرمجة والتكامل مع CI/CD.

CLI
ميزات واجهة الأوامر

يتصل عميل CLI بنسخة Aura Work عاملة عبر socket محلي. يستخدم نفس مخزن بيانات الاعتماد ويدعم كل المهارات والمزوّدين.

# ربط CLI بنسخة Aura Work عاملة
aura connect

# تشغيل مهمة
aura run "اكتب اختبارات لوحدة المصادقة"

# سير عمل قابل للبرمجة
aura run --script workflow.json

# إخراج JSON
aura run "حلل هذا الكود" --json

اوامر متقدمة

اوامر CLI المتقدمة

ادارة الجلسات

aura session list          # عرض الجلسات النشطة
aura session resume <id>  # استئناف جلسة سابقة
aura session kill <id>    # إنهاء جلسة

ادارة المهارات

aura skill list            # عرض المهارات المثبتة
aura skill install <name>  # تثبيت مهارة
aura skill create         # انشاء مهارة جديدة

ادارة MCP

aura mcp list              # عرض خوادم MCP
aura mcp add <name>       # اضافة خادم MCP
aura mcp remove <name>    # ازالة خادم MCP

المزامنة

aura sync push             # رفع التغييرات
aura sync pull             # تنزيل التغييرات
aura sync status           # حالة المزامنة

التكوين

aura config list           # عرض الاعدادات
aura config set <k> <v>   # تعيين اعداد
aura config reset          # اعادة للافتراضي
