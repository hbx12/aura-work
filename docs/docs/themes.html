Design System

Token architecture

The design system is defined in packages/ui/src/tokens.css with a layered architecture: primitives → semantic tokens → component tokens. All themes share the same primitive color ramps and override only semantic tokens.

🎨 Color primitives

A warm, clay-based palette with sand neutrals, a single terracotta accent, and semantic greens/ambers/reds. The accent shifts between light and dark modes while maintaining the same warm character.

📱 RTL & a11y ready

The system includes a high-contrast theme for accessibility, full RTL support for Arabic/Persian, and system-preference auto-detection via prefers-color-scheme.
