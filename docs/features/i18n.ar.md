# i18n — 20 لغة + Weblate

[![English](https://img.shields.io/badge/lang-en-blue.svg)](i18n.md)
[![العربية](https://img.shields.io/badge/lang-ar-green.svg)](i18n.ar.md)

## الغرض

دعم **20 لغة** في واجهة Aura Work مع RTL للعربية والفارسية، وملفات locale متوافقة مع Weblate.

## الملفات

| الملف | الوظيفة |
|-------|---------|
| `packages/i18n/src/catalog.ts` | الـ catalog المصدر + الترجمات |
| `packages/i18n/locales/*.json` | ملفات Weblate (flat JSON) |
| `packages/i18n/WEBLATE.md` | تعليمات Weblate |
| `apps/desktop/src-tauri/src/i18n.rs` | locale في SQLite + اكتشاف لغة النظام |
| `apps/desktop/src/hooks/useI18n.ts` | React hook + `dir=rtl` |

## اللغات (20)

English, Arabic, Spanish, French, German, Portuguese, Chinese (Simplified/Traditional), Japanese, Korean, Hindi, Indonesian, Turkish, Russian, Italian, Dutch, Polish, Vietnamese, Thai, Persian.

## السلوك

- لغة النظام إذا مدعومة → وإلا English
- المستخدم يغيّر اللغة من Settings
- `ar` و `fa` → `document.documentElement.dir = rtl`

## كيف تختبر

1. Settings → Language → العربية
2. تحقق من RTL في nav rail
3. `npm run build:locales -w @aura-os/i18n` → يحدّث `locales/*.json`

## Weblate

```bash
npm run build:locales -w @aura-os/i18n
```

ارفع `packages/i18n/locales/` إلى Weblate كـ JSON flat key-value.
