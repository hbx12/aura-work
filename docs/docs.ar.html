الوثائق

كل ما تحتاج معرفته عن آورا ورك — منصة وكلاء الذكاء الاصطناعي مفتوحة المصدر متعددة المزودين.

نظرة عامة

آورا ورك هي منصة وكلاء ذكاء اصطناعي لسطح المكتب، مفتوحة المصدر ومجانية، مبنية بـ Tauri 2 + React 19 مع خلفية Rust. تعمل محلياً على جهازك — بدون اعتماد على سحابة، بدون تتبع، وبدون حاجة لتسجيل.

تستخدم المنصة معمارية متعددة السايدكارات: كل قدرة رئيسية تعمل في عملية سايدكار خاصة بها (الوكيل، المتصفح، الآلة الافتراضية، المزامنة السحابية، إلخ) تتواصل عبر HTTP. هذا يجعل النظام معياريًا ومعزولًا ضد الأعطال وسهل التوسيع.

الفلسفة الأساسية: محلي أولاً، مصرّح به، قابل للتوسيع. أنت تملك بياناتك، مفاتيحك تبقى على جهازك، والوكيل يتصرف ضمن الحدود التي تحددها.

التثبيت

آورا ورك في مرحلة ألفا المبكرة. النسخ الجاهزة لأنظمة Windows وmacOS وLinux متاحة على صفحة الإصدارات.

للبناء من المصدر:

git clone https://github.com/hbx12/aura-work.git
cd aura-work
npm install
npm run tauri dev

نظام الصلاحيات

الإجراءات عالية التأثير تتطلب موافقة صريحة. نظام الصلاحيات يصنف الإجراءات حسب المخاطر ويطلب الموافقة قبل التنفيذ.

ملفات الصلاحيات المدمجة:

قراءة فقط

السماح بقراءة جميع الملفات.

أتمتة آمنة

قراءة وكتابة الملفات، أوامر قراءة الصدفة.

بحث

قراءة الملفات وتصفح الويب.

الخزنة المشفرة

مفاتيح API والأسرار مخزنة في خزنة مشفرة مدعومة من نظام التشغيل (Keychain في macOS، Credential Manager في Windows، Secret Service في Linux). الخزنة مربوطة بالجهاز وتدعم مفاتيح الاسترداد.

الجدولة

جدولة مهام ذكاء اصطناعي متكررة — اجتماعات يومية، تقارير أسبوعية، مراقبة كل ساعة. كل مهمة تعمل مع ملف الصلاحيات وسياسة التوجيه الخاصة بها.

المهارات

المهارات هي إعدادات وكيل ذكاء اصطناعي قابلة لإعادة الاستخدام — كل واحدة تحدد دوراً وإرشادات وتنسيق مخرجات لمهام محددة. يمكنك تثبيت المهارات من المتجر أو إنشاء مهاراتك الخاصة.

أدوات مخصصة

اكتب أدوات TypeScript الخاصة بك وشغّلها داخل الوكيل. الأدوات معزولة، يمكن اختبارها مباشرة، وتدعم التحقق من صحة الوسائط عبر JSON Schema. تتوفر قوالب: جلب API و أدوات الملفات.

خوادم MCP

آورا ورك تدعم بروتوكول سياق النموذج (MCP) بالكامل. يمكن ربط أي خادم MCP عبر نقل stdio أو SSE.

نظام الإضافات

الإضافات هي توسيعات مجمعة يمكنها إضافة أدوات وسايدكارات ومكونات واجهة مستخدم. إضافة Chrome (aura-chrome) مثال على ذلك.

معمارية السايدكارات

آورا ورك تشغل 8 سايدكارات، كل واحدة مسؤولة عن قدرة محددة. السايدكارات هي عمليات مستقلة تتواصل عبر HTTP:

Agent

تنسيق الوكيل الأساسي وتنفيذ المهام.

Bridge

يربط إضافة Chrome وإضافات Office.

Browser Helper

يدير أتمتة المتصفح.

Cloud Sync

مزامنة مشفرة بين الأجهزة.

Computer Use

يتحكم الوكيل بتطبيقات سطح المكتب.

Plugins Helper

يدير الإضافات وخوادم MCP والأدوات.

VM Helper

تنفيذ معزول في آلة افتراضية.
