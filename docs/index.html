Aura Work

Open-source, multi-provider desktop AI agent platform — local-first, permission-gated, self-hostable.

⭐ GitHub ⬇️ Download 📖 README

⚡ Features

🤖 Multi-Provider

OpenAI, Anthropic, Gemini, DeepSeek, Ollama — bring your own keys.

🔒 Local-First

Encrypted vault with OS-backed secure storage. No telemetry.

🛡️ Permission-Gated

High-impact actions require explicit approval. Full audit log.

🌍 20 Languages

Full RTL support for Arabic & Persian. Weblate-powered.

🧩 Extensible

Plugin system, MCP support, and official Marketplace.

🎨 17 Themes

Nord, Dracula, Cyberpunk, Sakura, and more.
