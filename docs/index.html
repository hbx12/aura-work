Features

Everything you need, nothing you don't

Local-first, permission-gated, and extensible. Aura Work puts you in control.

Multi-Provider

OpenAI, Anthropic, Gemini, DeepSeek, Ollama — bring your own keys or use local models. Route tasks to the best model.

Local-First

Encrypted vault with OS-backed secure storage. No telemetry. Your keys never leave your machine.

Permission-Gated

High-impact actions require explicit approval. Full audit log of every operation.

20 Languages

Full RTL support for Arabic and Persian. Weblate-powered community translations.

Extensible

Plugin system, MCP support, and official Marketplace. Build and share your own extensions.

17 Themes

Nord, Dracula, Cyberpunk, Sakura, and more. Light, dark, and AMOLED modes.

Everything you need to know about Aura Work.

Download the latest release from GitHub for Windows, macOS, or Linux. Run the installer and launch the app — no account or API key required to get started. Configure your first provider via Settings > Providers.
OpenAI, Anthropic (Claude), Google Gemini, DeepSeek, Ollama (local models), OpenRouter, Groq, Together AI, Mistral, and GitHub Copilot. The routing engine auto-discovers available providers and picks the best model per task.
Yes. Aura Work is local-first — your API keys are stored in an encrypted OS-backed vault, no telemetry is collected, and all agent execution happens on your machine. Use local models via Ollama for fully offline operation.
Yes, completely free and open source under the Apache 2.0 license. You only pay for API usage if you use cloud providers like OpenAI or Anthropic. Local models via Ollama cost nothing.
Sidecars are 8 modular background daemons that extend Aura Work's capabilities — agent execution, browser automation, VM sandboxing, cloud sync, computer-use, and more. Each runs independently and can be enabled or disabled from the Settings panel.
