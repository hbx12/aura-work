كل ما تحتاج معرفته عن Aura Work.

حمّل أحدث إصدار من GitHub لنظام ويندوز أو ماك أو لينكس. شغّل المثبّت وافتح التطبيق — لا تحتاج حساب أو مفتاح API للبدء. هيّئ أول مزوّد من خلال الإعدادات > المزوّدون.
OpenAI وAnthropic (Claude) وGoogle Gemini وDeepSeek وOllama (نماذج محلية) وOpenRouter وGroq وTogether AI وMistral وGitHub Copilot. محرك التوجيه يكتشف المزوّدين المتاحين تلقائياً ويختار أفضل نموذج لكل مهمة.
نعم. Aura Work يعمل محلياً أولاً — مفاتيح API مخزّنة في خزنة مشفّرة، لا توجد تتبع استخدام، وكل تنفيذ الوكيل يتم على جهازك. استخدم النماذج المحلية عبر Ollama للعمل بدون إنترنت.
نعم، مجاني بالكامل ومفتوح المصدر تحت رخصة Apache 2.0. تدفع فقط مقابل استخدام API إذا استخدمت مزوّدين سحابيين مثل OpenAI أو Anthropic. النماذج المحلية عبر Ollama لا تكلف شيئاً.
الخدمات المساعدة هي 8 برامج خلفية معيارية توسّع قدرات Aura Work — تنفيذ الوكلاء، أتمتة المتصفح، العزل في بيئة افتراضية، المزامنة السحابية، استخدام الكمبيوتر، والمزيد. يمكن تشغيل كل منها أو إيقافها من لوحة الإعدادات.
اطّلع على دليل المساهم الجديد والمشاكل المناسبة للبدء في GitHub. نرحب بالمساهمات البرمجية والتوثيق والترجمات عبر Weblate والتصميم والأفكار. انضم إلى المناقشات للتواصل مع المجتمع.
نعم — مع Ollama يمكنك تشغيل نماذج محلية بالكامل بدون أي اتصال بالإنترنت. المنصة الأساسية ومحرك التوجيه والخدمات المساعدة كلها تعمل بدون اتصال. فقط واجهات مزوّدي السحابة تتطلب اتصالاً.
