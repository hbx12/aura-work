وكيل ذكاء اصطناعي لسطح المكتب، محلي أولاً، متعدد المزودين. يدعم 10 مزوّدين مع توجيه ذكي، 18 مهارة متخصصة، أتمتة متصفح، بيئة افتراضية آمنة، ومزامنة سحابية مشفّرة طرف-إلى-طرف. كل شيء يعمل على جهازك — خصوصيتك أولاً.

10 مزوّدين

توجيه ذكي

5 سياسات: الجودة أولاً، التكلفة أولاً، الخصوصية أولاً، محلي فقط، أو يدوي. محرك التوجيه يختار أفضل نموذج لكل مهمة تلقائياً.

محلي أولاً

خزنة ChaCha20-Poly1305 + Argon2. لا تتبع. مفاتيحك لا تغادر جهازك أبداً. تشفير E2EE AES-256-GCM للمزامنة السحابية.

مصرّح به

3 مستويات صلاحية، 12 فئة صلاحية. الإجراءات عالية التأثير تتطلب موافقة صريحة. سجل تدقيق كامل لكل عملية.

20 لغة + 34 سمة

دعم كامل للعربية والفارسية مع RTL. ترجمات المجتمع عبر Weblate. 34 سمة: نورد، دراكيولا، سايبربانك، ساكورا والمزيد.

18 مهارة + سوق

وثائق، PDF، بحث، تصميم، تحليل بيانات، Git Master، وأكثر. السوق يتيح تثبيت مهارات مجتمعية لأي إطار عمل.

أتمتة المتصفح

متصفح Chromium مدمج مع دعم CDP كامل. التنقل، النقر، ملء النماذج، لقطات الشاشة، وتنفيذ JavaScript في بيئة معزولة.

استخدام الكمبيوتر

الوكلاء يتفاعلون مع سطح مكتبك — النقر، الكتابة، لقطات الشاشة، والتنقل في أي تطبيق. مثالي لاختبار واجهات المستخدم.

مهام مجدولة

أنشئ سير عمل متكررة — مراجعة كود تلقائية، تحديث توثيق، مراجعة تبعيات. تعمل عبر cron مدمج مع أي سياسة توجيه.

بيئة افتراضية آمنة

خدمات VM Sidecar تعزز تنفيذ الكود في بيئة معزولة. حماية SSRF وفحص حقن الأوامر المدمج.

مزامنة سحابية

تشفير طرف-إلى-طرف AES-256-GCM عبر الأجهزة. مشاريع، إعدادات، سجل تدقيق، ومهام مجدولة — كلها متزامنة بأمان.

MCP وملحقات

دعم بروتوكول سياق النموذج (MCP) مع نقل stdio/SSE/HTTP. نظام إضافات وسوق رسمي. ابنِ وشارك إضافاتك الخاصة.

البدء السريع

ابدأ في 3 خطوات

لا تحتاج حساب. لا تحتاج مفتاح API للاستكشاف. فقط حمّل وابدأ.

1. حمّل التطبيق

حمّل أحدث إصدار من GitHub لويندوز، ماك، أو لينكس. لا حاجة لحساب — فقط ثبّت وشغّل.

2. أنشئ مشروعاً

افتح مجلد كودك كمشروع. Aura يكتشف تلقائياً git، package.json، والإعدادات الموجودة.

3. ابدأ المهمة

اكتب طلبك بلغتك الطبيعية. الوكيل يقرأ ملفاتك، يفهم سياقك، وينفذ المهمة مع شرح كل خطوة.

Aura Cloud (مُدار، أرخص بمرتين)، OpenAI، Anthropic (Claude)، Google Gemini، DeepSeek، Ollama (نماذج محلية)، LM Studio (نماذج محلية)، Minimax، Qwen، وأي نقطة نهاية مخصصة. محرك التوجيه يكتشف المزوّدين المتاحين تلقائياً ويختار أفضل نموذج لكل مهمة بناءً على سياسة التوجيه.
نعم. Aura Work يعمل محلياً أولاً — مفاتيح API مخزّنة في خزنة مشفّرة، لا توجد تتبع استخدام، وكل تنفيذ الوكيل يتم على جهازك. استخدم النماذج المحلية عبر Ollama للعمل بدون إنترنت.
نعم، مجاني بالكامل ومفتوح المصدر تحت رخصة Apache 2.0. تدفع فقط مقابل استخدام API إذا استخدمت مزوّدين سحابيين مثل OpenAI أو Anthropic. Aura Cloud ي提供 مستوى مجاني، والنماذج المحلية عبر Ollama لا تكلف شيئاً.
الخدمات المساعدة هي 7 برامج خلفية معيارية توسّع قدرات Aura Work — تنفيذ الوكلاء، التواصل عبر الجسر، أتمتة المتصفح، العزل في بيئة افتراضية، المزامنة السحابية، استخدام компьютер، والملحقات. يمكن تشغيل كل منها أو إيقافها من لوحة الإعدادات.
اطّلع على دليل المساهم الجديد والمشاكل المناسبة للبدء في GitHub. نرحب بالمساهمات البرمجية والتوثيق والترجمات عبر Weblate والتصميم والأفكار. انضم إلى المناقشات للتواصل مع المجتمع.
نعم — مع Ollama يمكنك تشغيل نماذج محلية بالكامل بدون أي اتصال بالإنترنت. المنصة الأساسية ومحرك التوجيه والخدمات المساعدة كلها تعمل بدون اتصال. فقط واجهات مزوّدي السحابة تتطلب اتصالاً.
Aura Cloud هو خدمتنا المُدارة التي تختار تلقائياً أفضل نموذج لكل مهمة. أرخص بمرتين من أسعار API المباشرة، يقدم مكافأة 10% عند الشحن، ويتضمن مستوى مجاني للبدء. لا حاجة لإدارة مفاتيح API — فقط اتصل وابدأ.
المهارات هي وحدات ذكاء اصطناعي متخصصة — كل منها حزمة من الأدوات والأوامر التوجيهية وأفضل الممارسات. المكتبة المدمجة تتضمن 18 مهارة (وثائق، PDF، بحث، تصميم، تحليل بيانات، Git Master، والمزيد). السوق يتيح تثبيت مهارات مجتمعية.
خوادم MCP (بروتوكول سياق النموذج) تتيح لـ Aura الاتصال بأدوات خارجية مثل قواعد البيانات والواجهات البرمجية. قم بإعدادها لكل مشروع أو عالمياً من الإعدادات → الملحقات. Aura يدعم نقل stdio وSSE وHTTP مع حماية SSRF مدمجة.
يتضمن Aura متصفح Chromium مدمج مع دعم CDP كامل. يمكن للوكلاء التنقل في الصفحات، والنقر على العناصر، وملء النماذج، ولقطات الشاشة، وتنفيذ JavaScript — كل ذلك في بيئات معزولة مع حماية SSRF.
استخدام الكمبيوتر يتيح للوكلاء التفاعل مع سطح مكتبك — النقر، الكتابة، لقطات الشاشة، والتنقل في أي تطبيق. مدعوم بخدمة مساعدة مخصصة مع تكامل نماذج الرؤية. مفيد لاختبار واجهات المستخدم وأتمتة سير العمل.
مزامنة Aura Cloud توفّر مزامنة مشفرة طرف-إلى-طرف عبر أجهزتك باستخدام AES-256-GCM. بياناتك مشفرة قبل مغادرة جهازك — لا نرى مفاتيحك أو محادثاتك أو ملفاتك. تتضمن المشاريع والإعدادات وسجل التدقيق والمهام المجدولة.
نعم! نظام المهام المجدولة يتيح لك إنشاء سير عمل ذكاء اصطناعي متكررة — مراجعة كود تلقائية، تحديث توثيق، مراجعة تبعيات، والمزيد. تعمل عبر محرك cron مدمج مع أي سياسة توجيه.
ملفات تعريف الحيوانات الأليفة تتيح لك إنشاء شخصيات ذكاء اصطناعي مخصصة مع صور رمزية فريدة وأوامر توجيهية وتفضيلات سلوكية. أعطِ رفيقك البرمجي اسماً وشخصية ومظهراً يناسب أسلوبك. قم بإعدادها من الإعدادات → الحيوان الأليف.
